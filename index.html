Install the marketplace

Claude Code

CLI

claude plugin marketplace add nearform/skills

In Claude

/plugin marketplace add nearform/skills

Cursor

Open File → Cursor Settings → Plugins, then add nearform/skills.

Copilot CLI

copilot plugin marketplace add nearform/skills

Browse from your client too: /plugin in Claude Code · copilot plugin marketplace browse nearform-skills in Copilot CLI · the Plugins panel in Cursor.

Stay up to date

Claude Code

/plugin marketplace update nearform-skills

Cursor

Cursor refreshes the marketplace automatically. Force-refresh from Settings → Plugins.

Copilot CLI

copilot plugin marketplace update nearform-skills
